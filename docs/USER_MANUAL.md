## 1. 简介

TextDiff 是一款专业的文本对比工具，支持：

- **文件对比**: 并排双栏、统一视图展示文本差异
- **目录对比**: 递归对比文件夹，树形展示差异
- **三路合并**: 解决代码冲突，支持 Git 合并场景
- **忽略规则**: 灵活配置空白符、大小写、正则规则
- **多算法**: Myers / Patience / Histogram 算法可选
- **多平台**: Linux (AppImage/deb/rpm)、Windows、macOS
- **国际化**: 支持中文和英文界面
- **智能搜索**: 全局搜索差异内容，支持正则表达式
- **撤销/重做**: 完整的操作历史管理
- **差异缓存**: 智能缓存提升重复对比性能

## 2. 安装指南

### Linux

#### AppImage (推荐)
```bash
# 下载 AppImage
chmod +x TextDiff-1.0.0-x86_64.AppImage
./TextDiff-1.0.0-x86_64.AppImage
```

#### Debian/Ubuntu (.deb)
```bash
sudo dpkg -i textdiff_1.0.0_amd64.deb
sudo apt-get install -f  # 安装依赖
```

#### Fedora (.rpm)
```bash
sudo rpm -i textdiff-1.0.0.x86_64.rpm
```

#### 通用压缩包 (.tar.gz)
```bash
tar -xzf textdiff-1.0.0-x64.tar.gz
cd textdiff-1.0.0
./textdiff
```

### Windows

1. 下载 `TextDiff-1.0.0-setup.exe`
2. 双击运行安装向导
3. 或使用便携版 `TextDiff-1.0.0-portable.exe`

### macOS

1. 下载 `TextDiff-1.0.0.dmg`
2. 打开 DMG 文件，拖拽到 Applications 文件夹

## 3. 快速入门

### 3.1 启动应用

```bash
# GUI 模式
textdiff

# 带文件启动
textdiff file1.txt file2.txt
```

1. 打开应用后，点击 **"打开文件对"** 或按 `Ctrl+O`
2. 选择左侧文件（旧版本/基准版本）
3. 选择右侧文件（新版本/对比版本）
4. 查看差异高亮结果

### 3.3 拖拽文件

直接拖拽两个文件到应用窗口，自动识别为左右文件。

## 5. 文件对比

### 5.1 打开文件

- **菜单**: 文件 → 打开文件对 (`Ctrl+O`)
- **工具栏**: 点击"打开文件对"按钮
- **拖拽**: 拖拽两个文件到窗口
- **最近文件**: 从欢迎页或最近会话列表选择

### 5.2 视图模式

#### 双栏视图 (Split View) - `Ctrl+1`
- 左右两侧并排显示
- 同步滚动，便于对比
- 适合宽屏使用
- 集成 Monaco Editor，支持语法高亮和代码编辑

#### 统一视图 (Unified View) - `Ctrl+2`
- 单栏显示，左侧显示 `+`/`-` 标记
- 类似 `git diff` 输出格式
- 适合快速浏览

#### Monaco 编辑器视图
- 专业级代码编辑器支持
- 语法高亮支持 50+ 编程语言
- 代码折叠、 minimap 缩略图
- 直接编辑文件内容

### 5.3 差异导航

| 快捷键 | 功能 |
|--------|------|
| `F6` / `Alt+↑` | 上一处差异 |
| `F7` / `Alt+↓` | 下一处差异 |
| `Alt+Home` | 第一处差异 |
| `Alt+End` | 最后一处差异 |

### 5.4 搜索功能

按 `Ctrl+F` 打开搜索对话框：
- 支持普通文本搜索和正则表达式搜索
- 点击 `.*` 按钮切换正则模式
- `Enter` 跳转到下一个匹配项
- `Shift+Enter` 跳转到上一个匹配项
- `Esc` 关闭搜索框

搜索结果会在差异视图中高亮显示，状态栏显示匹配数量。

### 5.5 折叠相同区域

按 `Ctrl+Shift+C` 或点击工具栏"折叠相同"按钮，自动折叠连续的相同行，聚焦差异内容。

### 5.6 撤销与重做

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Z` | 撤销上一步操作 |
| `Ctrl+Y` / `Ctrl+Shift+Z` | 重做 |

支持的操作包括：
- 文件打开/关闭
- 忽略规则更改
- 视图模式切换

## 7. 三路合并

### 7.1 使用场景

解决 Git 合并冲突：
- **Base**: 共同祖先版本
- **Left (Current)**: 当前分支
- **Right (Target)**: 目标分支

### 7.2 操作流程

1. 打开三路合并视图 (`Ctrl+4`)
2. 分别选择 Base / Left / Right 文件
3. 查看冲突区域（红色高亮）
4. 选择解决方案：
   - **采用 Base**: 使用原始版本
   - **采用 Left**: 保留当前分支修改
   - **采用 Right**: 接受目标分支修改
   - **手动编辑**: 自定义合并内容

### 7.3 保存结果

合并完成后，点击"保存结果"将合并后的文件写入磁盘。

## 8. 忽略规则

### 8.1 快速忽略

工具栏提供三个快捷开关：
- **空白符**: 忽略空格、Tab 差异
- **大小写**: 忽略大小写差异
- **行尾符**: 忽略 CRLF / LF 差异

### 8.2 详细配置

点击工具栏"更多"或菜单 工具 → 忽略规则设置，打开详细配置面板：

#### 空白符处理
- **不忽略**: 精确对比
- **忽略首尾空白**: 忽略行首和行尾空格
- **忽略全部空白**: 完全忽略所有空白字符

#### 其他选项
- **忽略大小写差异**
- **忽略行尾符差异** (CRLF vs LF)
- **忽略注释行**: 可配置注释前缀 (`//`, `#`, `--` 等)

#### 自定义正则规则
添加正则表达式规则，匹配的行将被忽略。

示例：
- `^\s*#.*$` - 忽略 Python 注释行
- `^\s*//.*$` - 忽略 C++ 注释行
- `^\s*$` - 忽略空行

### 8.3 算法选择

| 算法 | 特点 | 适用场景 |
|------|------|---------|
| Myers | Git 同款默认算法 | 通用场景 |
| Patience | 更清晰的上下文 | 代码移动场景 |
| Histogram | 改进型 Patience | 大文件对比 |

## 9. 会话管理

### 9.1 保存会话

`Ctrl+S` 保存当前对比状态，包括：
- 左右文件路径
- 忽略规则设置
- 视图模式
- 当前滚动位置

### 9.2 会话历史

`Ctrl+H` 打开会话历史对话框：
- 查看已保存的会话列表
- 搜索历史会话
- 删除不需要的会话

### 9.3 标签页

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+T` | 新建对比标签 |
| `Ctrl+W` | 关闭当前标签 |
| `Ctrl+Tab` | 切换标签 |

### 9.4 界面语言

TextDiff 支持多语言界面：

**切换语言**：
1. 菜单 → 工具 → 语言
2. 选择 简体中文 或 English
3. 重启应用生效

或在欢迎页点击右上角语言切换按钮快速切换。

## 10. 搜索功能

TextDiff 提供强大的全局搜索功能，可在差异内容中快速定位目标文本。

### 10.1 打开搜索

- **快捷键**: `Ctrl+F`
- **菜单**: 编辑 → 搜索

### 10.2 搜索模式

#### 普通搜索
- 直接输入文本进行匹配
- 不区分大小写（可在设置中更改）

#### 正则表达式搜索
- 点击搜索框右侧的 `.*` 按钮启用
- 支持 JavaScript 正则语法
- 示例：`function\s+\w+` 匹配函数定义

### 10.3 搜索结果导航

| 操作 | 快捷键 |
|------|--------|
| 下一个匹配项 | `Enter` |
| 上一个匹配项 | `Shift+Enter` |
| 关闭搜索框 | `Esc` |

### 10.4 搜索范围

搜索范围包括：
- 左侧文件内容
- 右侧文件内容
- 新增、删除、修改的所有行

### 10.5 高亮显示

- 匹配项以黄色背景高亮
- 当前选中的匹配项以橙色边框标注
- 状态栏显示 `匹配 3/12` 形式的计数

## 12. 命令行工具

### 12.1 启动 GUI

```bash
# 启动 GUI
textdiff

# 启动 GUI 并打开文件
textdiff file1.txt file2.txt
```

### 12.2 文本对比

```bash
# 输出 unified diff 到 stdout
textdiff diff file1.txt file2.txt

# 输出到文件
textdiff diff file1.txt file2.txt -o output.diff

# 忽略空白符
textdiff diff file1.txt file2.txt --ignore-whitespace all

# 忽略大小写
textdiff diff file1.txt file2.txt --ignore-case
```

### 12.3 三路合并

```bash
# 三路合并
textdiff merge base.txt left.txt right.txt -o output.txt

# 自动合并，冲突时报错
textdiff merge base.txt left.txt right.txt -o output.txt --auto
```

### 12.4 帮助信息

```bash
textdiff --help
textdiff diff --help
textdiff merge --help
```

## 13. Git 集成

### 13.1 配置为 difftool

```bash
# 配置 Git 使用 TextDiff
git config --global diff.tool textdiff
git config --global difftool.textdiff.cmd 'textdiff diff "$LOCAL" "$REMOTE"'
git config --global difftool.prompt false

# 使用
git difftool HEAD~1
```

### 13.2 配置为 mergetool

```bash
# 配置 Git 使用 TextDiff 解决冲突
git config --global merge.tool textdiff
git config --global mergetool.textdiff.cmd 'textdiff merge "$BASE" "$LOCAL" "$REMOTE" -o "$MERGED"'
git config --global mergetool.prompt false
git config --global mergetool.keepBackup false

# 使用
git mergetool
```

### 13.3 临时使用

```bash
# 单次使用 TextDiff 对比
git difftool --tool=textdiff HEAD~1
```

## 14. 故障排除

### 14.1 应用无法启动

**问题**: 双击无反应  
**解决**:
- Linux: 检查 AppImage 是否有执行权限 `chmod +x`
- Linux: 安装依赖库 `libgtk-3-0`, `libnotify4`
- Windows: 检查是否被杀毒软件拦截

### 14.2 文件编码问题

**问题**: 中文显示乱码  
**解决**:
- 应用会自动检测文件编码
- 如检测错误，可在设置中手动指定默认编码

### 14.3 大文件性能

**问题**: 大文件对比卡顿  
**解决**:
- 超过 5MB 的文件会自动使用 Worker 线程处理
- 应用内置差异缓存，重复对比相同文件时速度更快
- 支持增量差异计算，小修改后重新对比性能优异
- 建议分割大文件后再对比
- 关闭"字符级差异"以提升性能

### 14.4 日志位置

```
Linux: ~/.config/textdiff/logs/
Windows: %APPDATA%\textdiff\logs\
macOS: ~/Library/Logs/textdiff/
```
